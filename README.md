# Photomaton

Aplicación sencilla en Python para simular el flujo de un photomaton:

1. Botón de "Realizar foto".
2. Cuenta atrás de 5 segundos, captura de 3 fotos con 2 segundos de intervalo.
3. Solicitud del número de teléfono para enviar las fotos por WhatsApp.
4. Pregunta para mostrar las fotos en pantalla (simulación de envío a una pantalla).

## Requisitos

- Python 3.x con `tkinter` disponible.

## Ejecución

```bash
python main.py
```
